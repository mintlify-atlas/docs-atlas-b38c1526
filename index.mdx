  <Steps>
    <Step title="Install the Extension">
      Download Docket from the [Chrome Web Store](http://awwscar.ca/docket/) or install it manually by loading the unpacked extension from the repository.
      
      <Note>
        Uninstalling and reinstalling will erase saved notes — back them up first!
      </Note>
    </Step>
    
    <Step title="Open a New Tab">
      Once installed, open a new tab in Chrome. Docket automatically replaces your default new tab page with a clean markdown editor.
      
      You can also click the Docket icon in your toolbar to open it as a popup.
    </Step>
    
    <Step title="Start Writing">
      Create your first note by clicking the **New Note** button. Write markdown in the editor pane, and watch it render live in the preview pane.
      
      ```markdown
      # Welcome to Docket
      
      - Create notes with **markdown**
      - Use `code blocks` for syntax highlighting
      - Organize notes by dragging them in the sidebar
      ```
    </Step>
    
    <Step title="Customize Your Experience">
      Toggle dark mode, choose a code highlighting theme, and switch between editor, split, or reader view modes using the toolbar.
    </Step>
  </Steps>
</div>

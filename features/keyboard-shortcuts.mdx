## Formatting Shortcuts

Accelerate your markdown formatting with these keyboard shortcuts:

### Bold Text

<Accordion title="Ctrl + B">
  Wrap selected text in bold formatting (`**text**`)
  
  ```typescript
  // Implementation: index.ts:818-827
  if (e.key === "b" && e.ctrlKey) {
    if (markdownInput.selectionStart !== markdownInput.selectionEnd) {
      e.preventDefault();
      markdownInput.setRangeText(
        "**" + markdownInput.value.substring(
          markdownInput.selectionStart, 
          markdownInput.selectionEnd
        ) + "**",
        markdownInput.selectionStart,
        markdownInput.selectionEnd,
        "select"
      );
    }
  }
  ```
</Accordion>

## Navigation Shortcuts

### Sidebar Toggle

<Accordion title="Ctrl + H">
  Toggle the sidebar visibility for a distraction-free editing experience
  
  ```typescript
  // Implementation: index.ts:720-724
  window.addEventListener("keydown", (e) => {
    if (e.key === "h" && e.ctrlKey) {
      e.preventDefault();
      toggleSidebar();
    }
  });
  ```
  
  The sidebar toggle function automatically adapts to screen size:
  
  ```typescript
  const toggleSidebar = () => {
    const sidebar = document.getElementById("sidebar");
    if ((!sidebar.style.display && window.innerWidth < 768) || 
        sidebar.style.display === "none") {
      openSidebar();
    } else {
      closeSidebar();
    }
  };
  ```
</Accordion>

## Editor Shortcuts

### Tab Indentation

<Accordion title="Tab">
  Insert a tab character at the cursor position (default tab behavior is overridden)
  
  ```typescript
  // Implementation: index.ts:804-816
  markdownInput.addEventListener("keydown", (e) => {
    if (e.key === "Tab") {
      e.preventDefault();
      if (markdownInput.selectionStart === markdownInput.selectionEnd) {
        markdownInput.setRangeText(
          "\t",
          markdownInput.selectionStart || 0,
          markdownInput.selectionEnd || 0,
          "end"
        );
      }
    }
  });
  ```
  
  <Info>
  The default tab behavior (focus switching) is disabled in the editor to allow proper indentation for code blocks and nested lists.
  </Info>
</Accordion>

## Implementation Details

All keyboard shortcuts are implemented in `index.ts` (lines 720-873) using event listeners on the markdown input element:

```typescript
// Editor event listeners setup
const markdownInput = document.getElementById("markdownInput");

markdownInput.addEventListener("keydown", (e) => {
  // Formatting shortcuts (bold, italic, links, etc.)
  // ...
  
  // Trigger re-render after formatting
  renderMarkdown();
  saveActiveNote();
});
```

<Note>
All formatting shortcuts automatically trigger markdown rendering and save operations, ensuring your changes are immediately visible and persisted.
</Note>

## Auto-Save Integration

Every keyboard shortcut triggers the auto-save mechanism:

```typescript
// index.ts:871-872
renderMarkdown();
saveActiveNote();
```

This ensures that:
1. Your formatted text is immediately rendered in the preview
2. Changes are automatically saved to IndexedDB
3. The note list is updated with the latest content
4. The `lastUpdated` timestamp is refreshed

<Warning>
Keyboard shortcuts only work when the markdown editor has focus. Click inside the editor area before using shortcuts.
</Warning>

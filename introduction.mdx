## What is Docket?

Docket is a Chrome extension that transforms your new tab page into a powerful markdown note-taking environment. All your notes are stored locally in your browser using IndexedDB, giving you fast, private access to your thoughts without any cloud dependencies.

<Note>
  Docket runs entirely in your browser - no server, no account, no data collection. Your notes stay on your device.
</Note>

## Why Use Docket?

Docket is perfect for developers and writers who:

- Need quick access to notes without leaving the browser
- Want markdown support with syntax highlighting for code snippets
- Prefer local-first applications that respect privacy
- Like minimal, distraction-free interfaces
- Use Chrome as their primary workspace

## Use Cases

### For Developers

- Maintain quick reference notes for APIs, commands, and code snippets
- Document debugging sessions with syntax-highlighted code
- Keep track of TODO lists and project notes
- Store commonly used bash commands or SQL queries

### For Writers & Students

- Draft blog posts and articles in markdown
- Take lecture notes with formatted text and code examples
- Maintain reading lists and research notes
- Create structured outlines and brainstorm ideas

### For General Productivity

- Replace your new tab page with actionable notes
- Quick capture of ideas without context switching
- Maintain personal wiki or knowledge base
- Track daily standup notes or meeting summaries

## Technical Details

Docket is built with:

- **Marked.js** - Markdown parsing with GFM support
- **highlight.js** - Code syntax highlighting
- **DOMPurify** - HTML sanitization for security
- **IndexedDB** - Local persistent storage
- **TypeScript** - Type-safe development
